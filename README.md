## Project Overview

This project analyzes Walmart's historical sales data to uncover business insights, identify sales trends, and build an interactive Power BI dashboard for decision-making. The analysis includes data cleaning, exploratory data analysis (EDA), visualization, and sales forecasting preparation.

The objective is to demonstrate practical data analytics skills using Python, Pandas, Matplotlib, Seaborn, and Power BI while solving real-world retail business problems.

---

## Dataset Information

**Dataset:** Walmart Store Sales Forecasting

The dataset contains historical weekly sales data across multiple Walmart stores and departments along with additional information such as:

- Store Number
- Department
- Weekly Sales
- Date

## Key Insights

Some important findings from the analysis:

- Holiday weeks generate approximately **7% higher average sales** than non-holiday weeks.
- A small number of stores contribute significantly to total revenue.
- Certain departments consistently outperform others across all stores.
- Weekly sales show strong seasonal patterns throughout the year.
- Business KPIs can be effectively monitored using an interactive Power BI dashboard.

## How to Run

### Clone the repository

```bash
git clone https://github.com/KiranKShettar/Financial-Data-Analytics-Forecasting.git
```

### Navigate to project

```bash
cd Financial-Data-Analytics-Forecasting
```

### Install dependencies

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open the notebooks in the following order:

1. Data Understanding
2. Data Cleaning
3. Exploratory Data Analysis

To explore the dashboard, open:

```
dashboard/Walmart_Sales_Dashboard.pbix
```

using Microsoft Power BI Desktop.
